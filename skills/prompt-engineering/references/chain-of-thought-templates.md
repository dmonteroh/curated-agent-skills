# Chain-of-Thought Templates

## Math template

```
Problem: {problem}

Steps:
1) Identify known values.
2) Identify unknowns.
3) Choose relevant formula.
4) Substitute values.
5) Compute and verify.

Final answer: {final_answer}
```

## Debugging template

```
Code:
{code}

Error:
{error}

Steps:
1) Interpret the error message.
2) Locate the failing line.
3) Explain the root cause.
4) Propose a fix.
5) Verify the fix.

Fixed code: {fixed_code}
```

## Logic template

```
Premises:
{premises}

Question: {question}

Steps:
1) List facts.
2) Identify relationships.
3) Apply deductions.
4) State conclusion.

Final answer: {final_answer}
```

## Evaluation checklist

- Each step is present and ordered.
- Calculations are shown before conclusions.
- Final answer is consistent with steps.
