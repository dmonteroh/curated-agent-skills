# Cloud and Infrastructure Security

## Cloud Security

- Cloud security posture management across AWS, Azure, and GCP.
- Infrastructure security: security groups, network ACLs, IAM policies.
- Data protection: encryption at rest/in transit with key management controls.
- Serverless security: least-privilege execution roles and event validation.
- Container security: Kubernetes Pod Security Standards and network policies.
- Multi-cloud security: consistent identity and policy enforcement.

## Network & Infrastructure Security

- Network segmentation and security zones.
- Firewall management for on-prem and cloud.
- Intrusion detection and prevention monitoring.
- VPN security with modern protocols.
- DNS security controls and monitoring.
