# Compliance and Governance

- Regulatory frameworks: GDPR, HIPAA, PCI-DSS, SOC 2, ISO 27001, NIST CSF.
- Compliance automation: policy as code, evidence collection, audit trails.
- Data governance: classification, privacy by design, residency requirements.
- Security metrics: KPIs, scorecards, and executive reporting.
- Incident response alignment: reporting, forensics, and notification workflows.
- See `references/compliance-checklist.md` for a condensed checklist.
